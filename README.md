# hotelManagement
Hotel Management Application 


Enhancements Added in Final Version :
Updated UI with CSS improvements, including a menu page redesigned with item pictures.
Added a Bill Generation & Payment feature for end-to-end order processing.
Implemented Reports, such as daily summaries.
Integrated Big Data analytics features, including:
– Real-time Orders Analytics (streaming dashboard).
– Popular Items Analytics based on frequency and demand.
Added a Kitchen Load Monitor to help visualize order volume and preparation load.
Incorporated Kafka for real-time data streaming and analytics across the system.
